# IB 동아리 — TOK 에세이·심포지엄 세특 워크북 (배포 완료)

> 사회문화·국제정치·BTS와 같은 패턴. **이미 배포 완료** — 아래 링크만 학생에게 주면 됨.

## 링크
- **학생용 워크북**: https://redtomatocat-del.github.io/culture/ib.html
- 수합 시트: https://docs.google.com/spreadsheets/d/1qcuIryXXFNKQtNo5QYva-Ze7VslbTJYr9hGeios4gU4/edit ("IB 동아리 세특")
- 학급 보드: 시트 메뉴 [🎓 IB 동아리 세특 → 학급 보드 열기]

## 구조 (계기 → 연구 → 결과 → 성찰·후속)
- STEP 1: 희망 전공·계열 / 에세이 제목 / **지식 영역(AOK)** 복수 선택 / **학습자상 10종** 복수 선택+직접 추가
- ① 동아리 활동 개요·역할 ② **지식 질문(KQ)** ③ 자료 수집·선별(출처·제외 기준) ④ 독서·개념 연결 ⑤ 분석(관점 비교·반례) ⑥ 에세이 논지(주장-반론-정당화) ⑦ 심포지엄 발표·토론(핵심역량 참고 내장) ⑧ 성찰 ⑨ 후속 탐구 예고
- 세특 초안 자동 생성: 제목 → 계기·역할 → 지식질문[AOK] → 자료 → 독서 → 분석 → 논지 → 심포지엄 → 성찰 → 후속 + TOK 탐구 사이클·학습자상 마무리 문장

## 운영
- 병합 저장: 같은 학년·반·번호·이름 재제출 시 빈 칸은 기존 값 보존
- 명단 없음: 제출한 학생만 마스터 탭에 표시, 미제출은 명렬표 대조
- 학생 안내 예시: "학년·반·번호·이름 정확히 → ② 지식 질문이 심장 → ③ 출처·선별 기준과 ⑦ 질의응답이 세특 뼈대 → 여러 번 제출 OK"

## 학생 안내문 (복붙용)
```

👉 https://redtomatocat-del.github.io/culture/ib.html
1) 학년·반·번호·이름 정확히 입력
2) 에세이 초고·독서 메모·심포지엄 기록을 펴 놓고 옮기기
3) ② 지식 질문(KQ)은 "~은 어느 정도까지 ~인가" 열린 질문으로
4) ③ 자료 출처·선별 기준 + ⑦ 심포지엄 질의응답이 세특의 뼈대
5) 제출 후 추가 제출해도 먼저 쓴 내용은 보존됨
마감: [날짜]
```
